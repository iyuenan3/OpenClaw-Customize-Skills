# 故事创作者

为番茄小说平台创作短篇小说的专业技能。

## 创作规范

**必达要求**：
- 字数：7000-10000字（不得低于7000字）
- 格式：TXT纯文本
- 保存位置：`workspace/story/`
- 命名：`YYYY-MM-DD-HH-MM-小说名称.txt`

## 创作流程

1. **确认题材** - 判断用户是否提供题材：
   - **用户提供题材** → 直接使用用户题材，跳过搜索
   - **用户未提供题材** → 搜索热门题材并自动选择
   - 搜索关键词：`番茄小说 短故事 热门题材 2026` 或 `番茄小说 爆款短篇 最新`
2. **参考指南** - 阅读创作技巧：
   - 写作风格：[writing-guide.md](references/writing-guide.md)
   - 热门题材：[genres.md](references/genres.md)
   - 标题公式：[title-formula.md](references/title-formula.md)
3. **制定大纲** - 规划章节结构与字数分配：
   - 确定100字导语（钩子）
   - 规划章节数量（通常5-8章）
   - 分配每章字数（确保总字数≥7000）
   - 示例分配：
     - 导语：100字
     - 第1章（开篇冲突）：1000-1500字
     - 第2章（冲突升级）：1000-1500字
     - 第3章（高潮/转折）：1500-2000字
     - 第4章（冲突激化）：1000-1500字
     - 第5章（结局收尾）：1000-1500字
4. **撰写正文** - 一次性完成7000字以上
5. **保存文件** - 按命名规范保存
6. **自动审查** - 检查质量标准：
   - 字数检查：运行 `python3 scripts/word_count.py <文件路径>`
   - 重复检测：运行 `python3 scripts/check_duplicates.py <文件路径>`
   - 内容审查：对照下方审查清单逐项检查
7. **审查结果处理**：
   - ✅ **通过**：向用户汇报完成，展示标题、分类、字数
   - ❌ **不通过**：说明问题 → 修改正文 → 重新审查 → 循环直到通过

## 审查循环保障

**最大重试次数**：3次
- 第1次不通过 → 针对问题修改
- 第2次不通过 → 重新审视大纲，调整方向
- 第3次不通过 → 人工介入，向用户汇报情况

**问题追踪**：
- 每次审查记录具体问题点
- 修改时逐一对照问题清单
- 确保每个问题都被处理

**渐进式处理**：
- 第1次：针对性修补（修复具体问题点）
- 第2次：结构性调整（重新审视章节安排）
- 第3次：人工决策（继续/换题材/重新创作）

## 审查清单

### 一、基础检查
- [ ] 字数 ≥ 7000字
- [ ] 无重复段落（重复率 < 5%）
- [ ] 格式正确（标题 + 正文 + 完结标注）

### 二、章节检查
**每章开篇**：
- [ ] 第1章开篇：100字内抛出核心矛盾/悬念
- [ ] 中间章节开篇：承接上文，快速进入场景
- [ ] 每章开篇有吸引力，不拖沓

**每章结尾**：
- [ ] 每章结尾有钩子（悬念/反转/冲突升级）
- [ ] 读者有继续阅读的冲动
- [ ] 不使用"欲知后事如何"等陈旧套路

**章节节奏**：
- [ ] 章节长度适中（1000-2000字/章）
- [ ] 无明显拖沓章节
- [ ] 每章有明确推进（人物/情节/情绪）

### 三、对话与叙事检查
**对话比例**：
- [ ] 对话占比合理（30%-60%为宜）
- [ ] 对话推动情节，非闲聊
- [ ] 对话符合人物性格和身份

**叙事平衡**：
- [ ] 旁白/描写不冗长（单段不超过150字）
- [ ] 无大段心理独白（超过300字需拆分）
- [ ] 行为描写与心理描写平衡

**对话质量**：
- [ ] 无"说"以外的动词引出对话
- [ ] 无副词修饰"说"
- [ ] 对话自然，不书面化

### 四、情绪曲线检查
**起伏节奏**：
- [ ] 开篇有强情绪点（愤怒/震惊/悬念）
- [ ] 前30%有第一个高潮/转折
- [ ] 中段有二次冲突/升级
- [ ] 后段有最强高潮戏
- [ ] 结局有情绪释放/反转

**情绪层次**：
- [ ] 不只有单一情绪（如全程压抑或全程爽）
- [ ] 情绪有递进（弱→强→最强）
- [ ] 有情绪缓冲区（紧张后给喘息空间）

**代入感检查**：
- [ ] 读者能代入主角视角
- [ ] 情绪反应符合人物设定
- [ ] 无跳出现象（突然OOC）

### 五、内容质量检查
**开篇质量**：
- [ ] 开篇100字内直入主题
- [ ] 前300字有核心矛盾展现
- [ ] 开篇不写天气、不写环境描写

**人物塑造**：
- [ ] 人物性格鲜明，行为合理
- [ ] 人物有记忆点（行为/口头禅/特征）
- [ ] 配角不抢戏，功能性明确

**情节结构**：
- [ ] 矛盾冲突强烈，有起伏
- [ ] 无逻辑漏洞
- [ ] 反转意外但合理

**结局质量**：
- [ ] 结局完整，解决开篇矛盾
- [ ] 首尾呼应，伏笔回收
- [ ] 不仓促，不烂尾

### 六、合规检查
- [ ] 无涉黄、涉毒、涉黑、暴恐内容
- [ ] 无性暗示、违背伦理描写
- [ ] 无抄袭、洗稿嫌疑
- [ ] 无敏感政治内容

### 不合格处理对照表

| 问题类型 | 处理方式 |
|---------|---------|
| 字数不足 | 扩写情节细节，增加对话和场景 |
| 重复过多 | 替换重复段落为原创内容 |
| 开篇平淡 | 重写前100字，制造悬念/冲突 |
| 章节开篇无钩子 | 增加悬念或冲突点 |
| 章节结尾无钩子 | 添加反转或悬念收尾 |
| 对话占比过低 | 增加人物对话，减少旁白 |
| 对话占比过高 | 增加行为描写和心理刻画 |
| 情绪平淡无起伏 | 增加冲突强度，调整节奏 |
| 缺少高潮 | 增加/强化冲突戏份 |
| 结局仓促 | 补充收尾情节，回收伏笔 |
| 人物OOC | 重新审视人物设定，调整行为 |
| 违规内容 | 立即删除重写相关段落 |

## 投稿红线

**绝对禁止**：涉黄、涉毒、涉黑、暴恐、血腥、敏感政治、诱导自残
**禁止擦边**：性暗示描写、违背伦理道德
**禁止抄袭**：洗稿、抄袭他站作品

## 核心技巧

- **开篇100字**：直入主题，制造极强看点
- **前30%**：设置最强烈剧情解锁点
- **黄金7步**：导语抓眼 → 行为塑人 → 首冲突 → 细节升温 → 二次冲突 → 高潮戏 → 完整结局
- **心法**：情绪>情节，细节>概括，克制>泛滥
